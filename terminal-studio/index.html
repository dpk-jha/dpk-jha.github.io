The Problem

Six apps open.
One workflow.

You run a command. Check git status in another window. Open a file in your editor. Browse the directory somewhere else. Every alt-tab is a context switch. Every context switch costs focus.

Terminal Studio — all of the above, one window
Why Terminal Studio

Built for how you actually work.

Not another terminal with a theme picker. A complete environment for developers who manage multiple projects and hate context switching.

Color-Coded Workspaces

Assign colors to project directories. Terminal Studio auto-activates the right workspace when you cd in. Multi-project chaos, visually organized.

Everything Built In

Git diff, stage, commit, push. File browser with .gitignore support. Syntax-highlighted editor. Markdown preview. Zero plugins. Zero setup.

GPU-Accelerated

Rust + wgpu. 60fps text rendering. Million-line scrollback without stutter. Your terminal shouldn't be what slows you down.

Session Persistence

Close it. Reopen it. Everything's back — sessions, scrollback history, workspace state. Resume exactly where you left off.

Performance

Rust. Not Electron.

Every architectural decision in Terminal Studio is about speed. Native rendering, dedicated threads, zero garbage collection pauses. Your terminal should be the fastest app on your machine.

GPU-Accelerated Rendering

wgpu renders text at 60fps. No DOM. No virtual scrolling hacks. Raw GPU power for text.

Per-PTY Reader Threads

Every terminal session gets its own reader thread. Heavy output in one pane doesn't block the others.

Async Everything

File browsing, git operations, search, system monitoring — all background threads. The UI thread stays free.

Alacritty's Parser

Uses the same VTE/ANSI parser as Alacritty. Battle-tested terminal emulation you can trust.
